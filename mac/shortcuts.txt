close window: cmd + w

SAFARI:
switch tabs: cmd + shift + arrow
new tab: cmd + t
close tab: cmd + w
open console: option + cmd + c (develop tab must be enabled)

open terminal: ctrl + `
command palette: shift + cmd + p
go to definition: fn + f12
peek definition: fn + option + f12
show suggestions: ctrl + space
rename: fn + f2
fold all functions: cmd + k + 0
move to neighbor tab: alt + cmd + arrow
move between tabs (listed): ctrl + tab
